# Politique de Confidentialité - Music Box

## Introduction

Music Box ("nous", "notre" ou "l'application") respecte votre vie privée et s'engage à protéger vos données personnelles. Cette politique de confidentialité vous informe sur la manière dont nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application de lecteur de musique.

## Informations que nous collectons

### Données locales
- **Bibliothèque musicale** : L'application accède à vos fichiers audio locaux pour créer votre bibliothèque musicale. Ces données restent uniquement sur votre appareil.
- **Métadonnées** : Titres, artistes, albums, pochettes et autres informations de vos fichiers audio.
- **Préférences utilisateur** : Vos playlists, favoris, historique d'écoute et paramètres de l'application.
- **Pochettes personnalisées** : Les images que vous choisissez pour personnaliser vos chansons.

### Données de publicité
L'application utilise Google AdMob pour afficher des publicités. AdMob peut collecter :
- Identifiant publicitaire Android (AAID)
- Adresse IP
- Type d'appareil et système d'exploitation
- Interactions avec les publicités

Pour plus d'informations sur les pratiques de Google : [Politique de confidentialité Google](https://policies.google.com/privacy)

### Données analytiques
Nous utilisons Firebase Analytics pour améliorer l'application :
- Statistiques d'utilisation anonymes
- Rapports de plantage
- Données de performance

## Comment nous utilisons vos données

- **Fonctionnement de l'application** : Pour lire votre musique locale, gérer vos playlists et personnaliser votre expérience.
- **Amélioration du service** : Pour corriger les bugs et améliorer les fonctionnalités.
- **Publicités personnalisées** : AdMob peut utiliser vos données pour afficher des publicités pertinentes.

## Stockage des données

- **Stockage local uniquement** : Toutes vos données musicales, playlists et préférences sont stockées localement sur votre appareil.
- **Aucun serveur distant** : Nous ne transférons ni ne stockons vos données personnelles sur nos serveurs.
- **Pochettes personnalisées** : Stockées dans le dossier de l'application sur votre appareil.

## Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers, sauf :
- **Google AdMob** : Pour afficher des publicités (voir leur politique de confidentialité)
- **Google Firebase** : Pour les analyses anonymes
- **Exigences légales** : Si requis par la loi

## Vos droits

Vous avez le droit de :
- Accéder à vos données (directement dans l'application)
- Supprimer vos données (en désinstallant l'application ou via les paramètres)
- Refuser les publicités personnalisées (dans les paramètres Android → Google → Annonces)

## Permissions de l'application

Music Box demande les permissions suivantes :
- **LECTURE_STOCKAGE_EXTERNE** : Pour accéder à votre musique
- **INTERNET** : Pour afficher les publicités et envoyer les analyses
- **MODIFIER_PARAMÈTRES_AUDIO** : Pour l'égaliseur système
- **PREMIER_PLAN_SERVICE** : Pour la lecture en arrière-plan
- **IGNORER_OPTIMISATIONS_BATTERIE** : Pour une lecture ininterrompue

## Sécurité

Nous prenons des mesures raisonnables pour protéger vos données, mais aucune transmission sur Internet n'est 100% sécurisée. Vos données musicales restent sur votre appareil et ne sont jamais envoyées à nos serveurs.

## Services tiers

L'application utilise des services tiers qui ont leurs propres politiques de confidentialité :
- [Google AdMob](https://support.google.com/admob/answer/6128543)
- [Firebase Analytics](https://firebase.google.com/support/privacy)

## Enfants

Music Box ne collecte pas sciemment d'informations personnelles d'enfants de moins de 13 ans. Si vous êtes un parent et découvrez que votre enfant nous a fourni des données, contactez-nous.

## Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité périodiquement. Nous vous notifierons de tout changement en publiant la nouvelle politique dans l'application et en mettant à jour la date de "Dernière mise à jour".
